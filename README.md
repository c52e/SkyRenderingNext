# SkyRenderingNext

Path traced cloud and atmosphere rendering in D3D12.

## Requirements

* Windows 10/11
* Visual Studio 2022

## Build

```
git clone git@github.com:c52e/SkyRenderingNext.git --single-branch
```
Open SkyRenderingNext.sln.

Build and run.
